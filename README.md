# Wifi-Pineapple-Pager-Payloads
A collection of payloads I have made for the wifi pineapple pager

# File Structure
```
/root/payloads/user - Payloads from payload screen
/root/payloads/alerts - Payloads for alert generation
/root/payloads/recon/access-point - Payloads specifically for APs in the recon screen
/root/payloads/recon/client - Payloads specifically for clients in the recon screen
```
